# Alteração de Preço


Programa criado para auxiliar no meu ambiente de trabalho na alteração de preços de itens próximos da validade que tem promoção de leve e ganhe. (promoção a cada 3/5/etc.)


O programa recebe dados como quantidade de itens a serem calculados, quantidade de itens que entram na promoção, valor dos itens, desconto e etc. E mostra para quanto a unidade deve ser alterada no PDV, já considerando o desconto da promoção.
